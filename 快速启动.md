## 🚀 启动服务

### Windows 用户

1. **启动后端**（必需）
```powershell
start.bat
```
或手动运行：
```powershell
cd "d:\工作\城建院\2603\B站信息检索\bilibili_search"
python -m uvicorn api.main:app --reload --host 0.0.0.0 --port 8000
```

2. **启动前端**（推荐）
```powershell
cd web
npm run dev
```

### Linux/Mac 用户

1. **启动后端**
```bash
cd /path/to/bilibili_search
python3 -m uvicorn api.main:app --reload --host 0.0.0.0 --port 8000
```

2. **启动前端**
```bash
cd web
npm run dev
```

## ⚠️ 注意事项

1. **后端必须先启动**，否则前端会报 `ERR_CONNECTION_REFUSED` 错误
2. 确保8000端口未被占用
3. 首次运行会自动安装依赖
4. 如需AI总结功能，请在 `.env` 文件中配置 `LLM_API_KEY`

## 🛠️ 故障排除

### 后端无法启动
```powershell
# 检查端口占用
netstat -ano | findstr :8000

# 终止占用进程
taskkill /PID <进程ID> /F
```

### 前端无法访问
```bash
cd web
npm install
npm run dev
```

### API连接失败
```
1. 确认后端已启动（访问 http://localhost:8000/docs）
2. 检查浏览器控制台（F12）的错误信息
3. 查看 `docs/错误排查指南.md`
```
