# 一键启动指南

本项目是一个完整的 B站视频搜索与AI总结系统，只需运行 `start.bat`（Windows）或 `start.sh`（Linux/Mac）即可一键启动。

## 📋 功能清单

### 核心功能
- ✅ **B站视频搜索**：关键词搜索、多页浏览、排序
- ✅ **AI视频总结**：基于字幕提取，使用智谱AI生成内容总结
- ✅ **综合视频推送**：多关键词搜索、智能筛选、生成Markdown推送
- ✅ **视频详情获取**：获取视频完整信息和统计数据
- ✅ **图片代理**：绕过B站防盗链，直接显示视频封面

## 🚀 快速启动

### Windows 用户

```bash
# 双击运行
start.bat
```

### Linux/Mac 用户

```bash
# 给予执行权限
chmod +x start.sh

# 运行
./start.sh
```

启动脚本会自动：
1. 检查Python环境（要求3.8+）
2. 检查Node.js环境（要求18+，可选）
3. 安装/更新后端依赖
4. 安装/更新 yt-dlp（AI总结需要）
5. 检查/创建 .env 配置文件
6. 验证B站搜索模块
7. 验证AI总结模块
8. 启动后端服务（端口8000）
9. 启动前端服务（端口5173，如果有Node.js）

## 📝 配置说明

首次运行前，建议配置 `.env` 文件：

```env
# B站配置（可选）
BILIBILI_SESSDATA=

# AI总结配置（可选，但推荐配置）
LLM_API_KEY=你的智谱API_KEY
LLM_API_BASE=https://api.deepseek.com/v1
LLM_MODEL=deepseek-chat
```

### 获取 B站 Cookie（可选）

如果需要登录态搜索，可以按以下步骤获取：

1. 浏览器打开 [B站](https://www.bilibili.com)
2. 登录后按 F12 打开开发者工具
3. 切换到 `Application` 标签 → `Cookies`
4. 找到 `SESSDATA` 字段并复制值

### 获取智谱AI Key（可选，但推荐）

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/usercenter/apikeys)
2. 注册并登录
3. 创建新的API Key
4. 复制API Key到 `.env` 文件

## 📖 使用示例

### B站搜索
```bash
# 基本搜索
curl "http://localhost:8000/api/v1/search?keyword=Python&platform=bilibili&page=1&page_size=10&order=totalrank"

# 搜索第二页
curl "http://localhost:8000/api/v1/search?keyword=编程&page=2&page_size=20"
```

### AI视频总结
```bash
# 总结单个视频
curl -X POST "http://localhost:8000/api/v1/summarize?url=https://www.bilibili.com/video/BV1et411s7eU"
```

### 综合视频推送
```bash
# 基本推送
curl -X POST "http://localhost:8000/api/v1/digest?keywords=Python&keywords=编程&per_keyword=5&max_videos=15"

# 高级推送（播放量筛选）
curl -X POST "http://localhost:8000/api/v1/digest?keywords=教程&keywords=实战&min_play=50000&max_videos=20"

# 按点赞排序
curl -X POST "http://localhost:8000/api/v1/digest?keywords=React&keywords=Vue&sort_by=like&max_videos=10"
```

## 🔧 依赖说明

### Python 依赖（自动安装）

项目会自动安装以下Python包：

- `fastapi` - Web框架
- `uvicorn` - ASGI服务器
- `pydantic` - 数据验证
- `pydantic-settings` - 配置管理
- `python-dotenv` - 环境变量加载
- `httpx` - 异步HTTP客户端
- `yt-dlp` - 视频字幕提取

### Node.js 依赖（自动安装）

前端会自动安装 npm 包，包括：

- `react`、`react-router-dom` - React框架
- `@tailwindcss/vite` - 样式框架
- `lucide-react` - 图标库
- `axios` 或 `fetch` - HTTP请求

## ⚠️ 常见问题

### 问题1：Python版本过低
```
错误：未找到Python，请先安装Python 3.8+
解决：升级Python到3.8或更高版本
```

### 问题2：端口被占用
```
错误：Address already in use
解决：
1. Windows：在任务管理器中关闭占用端口的进程
2. Linux/Mac：使用 `lsof -i:8000` 查找并终止进程
```

### 问题3：依赖安装失败
```
错误：pip install 失败
解决：
1. 使用国内镜像：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名`
2. 升级pip：`python -m pip install --upgrade pip`
```

### 问题4：前端无法启动
```
错误：未找到Node.js，将跳过前端启动
解决：
1. 安装Node.js：https://nodejs.org/
2. 如果只需要后端API，可以忽略前端
```

## 🎯 性能优化建议

### 后端优化

1. **生产环境部署**
   ```bash
   # 使用多worker模式
   uvicorn api.main:app --workers 4 --host 0.0.0.0 --port 8000
   ```

2. **配置Nginx反向代理**（可选）
   ```nginx
   location /api/ {
       proxy_pass http://localhost:8000;
       proxy_set_header Host $host;
       proxy_set_header X-Real-IP $remote_addr;
   }
   ```

### 前端优化

1. **构建生产版本**
   ```bash
   cd web
   npm run build
   ```

2. **部署到静态服务器**
   将 `web/dist` 目录部署到Nginx、Apache等
